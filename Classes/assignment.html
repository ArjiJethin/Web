Understanding Various Text Tags

In HTML there are two tags used to bold the text, they are Bold Tag and the other is Strong Tag. The main difference between Bold Tag and Strong Tag is b tag is used for styling purpose whereas the strong tag is used for telling the user that's an important block which we shouldn't miss.
The other two similar tags in HTML are Italic or I Tag and Emphasis Tag, even here the main difference between these two tags is same as the above case where i tag or Italic tag is used for stylings whereas, the Emphasis tag is used to emphasize the text and makes the user look it.

The purpose of mark tag is to make the text background color to highlighted yellow color which grabs the attention of the user. The del tag stands for delete and it is used to strike over a incorrect text like this. And similarly, the ins tag stands for inserted text, it is used make the text underline, which looks like this corrected. The sub tag stands for subscript which is used for the text or formulas which contain base values like: H 2O. The sup tag stands for superscript and it can be used for the power value in formulas for example X 2 or even for 3 rd sem.
